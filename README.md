# Members Only

A full-stack message board application built with **Node.js, Express, PostgreSQL, and EJS**. The app allows users to sign up, log in, and create posts, with special privileges for members and admins.

## Features

- View a list of **all posts**
- **User authentication** with login and signup
- **Create and delete posts**
- **Password hashing** using bcrypt
- **Session-based authentication** using Passport.js
- **Role-based access control**:
  - Members can see post authors and dates
  - Admins can delete posts
- **Join membership** and **become admin** via secret codes
- **Server-side validation** using express-validator
- **PostgreSQL relational database**
- **MVC-style architecture** with controllers, models, routes, and views
- Dynamic UI built using **EJS templates and CSS**
- Code for member is "member123", code for admin is in a similar format (try to guess it)

## Technologies Used

- **Node.js**
- **Express.js**
- **PostgreSQL**
- **EJS** templating engine
- **Passport.js** for authentication
- **bcryptjs** for password hashing
- **express-validator** for input validation
- **express-session** for session management
- **dotenv** for environment variables
- **CSS3** for styling
